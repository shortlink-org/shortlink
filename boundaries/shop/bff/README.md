# WunderGraph simple example

#### Getting started

```shell
npmp i && npmp start
```

#### Get all Continents

```shell
curl http://localhost:9991/operations/Continents
```

#### Get all Countries

```shell
curl http://localhost:9991/operations/Countries
```

#### Get user

```shell
curl http://localhost:9991/operations/users/get?id=1
```
