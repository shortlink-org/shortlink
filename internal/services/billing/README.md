# Billing

Service for work with billing.

**Functionality**:

  * CRUD operations for payment
  * SQRS operations for payment
  * change tariff

### ADR

- [ADR-0001](./docs/ADR/decisions/0001-init.md) - Init project

- UC-5 - Change tariff
